## ğŸ¨ Key Features

### 1. **Modular LOB Isolation**
- Each LOB has independent:
  - Routes (`app/{lob}/page.tsx`)
  - Components (`components/{lob}/`)
  - Logic (`lib/{lob}/`)
  - State management (`lib/{lob}/store.ts`)

### 2. **Shared Core Engine**
- Conversation engine (`lib/core/`)
- Store factory (`lib/core/createStore.ts`)
- Base types (`lib/core/types.ts`)
- Shared components (`components/core/`)

### 3. **Progressive Disclosure**
- Information collected step-by-step
- No overwhelming forms
- Context-aware questions

### 4. **Persona-Based Personalization**
- Dynamic persona detection
- Personalized messaging
- Tailored recommendations

### 5. **Multi-Language Support**
- English, Hindi, Hinglish, Kannada
- Per-LOB translations
- Dynamic language switching
